1. What is ReactJS?
ReactJS is a JavaScript library used to build reusable components for the view layer in MVC architecture. 
It is highly efficient and uses a virtual DOM to render components. It works on the client side and is written in JSX.

2. Explain the MVC architecture?
The Model-View-Controller (MVC) framework is an architectural/design pattern that separates an application into 
three main logical components Model, View, and Controller. Each architectural component is built to handle specific 
development aspects of an application. It isolates the business, logic, and presentation layer from each other
